# 📚 AI-Research Paper Summarizer 

A Python-based AI research paper summarizer built using LangChain, OpenAI's `gpt-4o-mini` model, and a Streamlit web interface. The tool summarizes selected research papers according to user-defined explanation style and desired summary length. This project demonstrates how LLM-powered tools can make complex academic papers easier to understand.

---

## 🚀 Features

- 📄 Summarizes well-known research papers using AI
- 🎚 Adjustable summary length
- 🧠 Customizable explanation style
- 🌐 Simple Streamlit web interface
- ⚡ Powered by LangChain + OpenAI GPT models
- 🔑 Uses your OpenAI API key

---

## 🧾 Supported Research Papers

Currently, the tool can summarize the following papers:

- Attention is All You Need
- BERT: Pre-training of Deep Bidirectional Transformers for Language Understanding
- GPT-3: Language Models are Few-Shot Learners
- Diffusion Models Beat GANs on Image Synthesis
- The AAA+ Chaperone VCP Disaggregates Tau Fibrils and Generates Aggregate Seeds

These papers are currently preloaded into the system.

## 🏗 Project Architecture

```
User Input (Streamlit UI)
        │
        ▼
Custom Settings
(summary length + explanation style)
        │
        ▼
LangChain Pipeline
        │
        ▼
OpenAI Model (gpt-4o-mini)
        │
        ▼
Generated Summary
```

## 📦 Dependencies

The project uses the following Python packages:

```
langchain
streamlit==1.53.1
langchain-openai==0.3.18
dotenv==0.9.9
pip==25.2
```

Install them with:

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Setup

Create a `.env` file in the project root:

```
OPENAI_API_KEY=your_openai_api_key_here
```

The application uses this key to access the OpenAI chat model.

---

## ▶️ Running the Application

Start the Streamlit app with:

```bash
streamlit run Research_paper_summarizer.py
```

Then open the local URL displayed in the terminal (usually):

```
http://localhost:8501
```
